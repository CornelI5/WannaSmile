WannaSmile Languack
A super simple oh gosh programming language, minimalist programming language for developers who don't have time for semicolons and bloated syntax.

Why WannaSmile?
- **Minimalist:** No unnecessary noise.
- **idk:** Logic-first, syntax-second.
- **Agentic:** Built to interact with system processes natively.
